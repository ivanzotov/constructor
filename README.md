## Installation

### Add to Gemfile

    gem 'constructor-cms', :git => 'https://github.com/raizstudio/constructor.git', :branch => "1.1"

### Install

    bundle install

### Migrate
    
    rake railties:install:migrations
    
    rake db:migrate
